# AI Usage

I used Anthropic's Claude through claude.ai while building this project. Here's what that actually looked like:

- Talking through design and architecture decisions, and clarifying parts of the brief that were ambiguous
- Drafting and reviewing prose in the markdown files in this repo
- Reviewing code I had written and arguing through implementation tradeoffs
- Walking through how to build certain components, which I then wrote myself
- A handful of cases where Claude drafted a code block, which I read line by line, edited where I disagreed, and integrated by hand

No coding agent with filesystem access to this repo was used at any point. That includes Claude Code, Cursor agent mode, GitHub Copilot agent mode, Aider, and Devin. None of them ran against this repo.

Every commit was made by me after reviewing what was going in. I take full responsibility for the correctness and design of all code here. Build, test, and run commands are in README.md.
